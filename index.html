Klädkod Dress Code

För herrar en hel kostym dvs. byxor som matchar kavajen. Kan vara mörk eller ljus med skorta tillsammans med slips eller fluga i valfri färg. För damer strax över knälång klänning eller längre, slits ovanför knäna går bra. Går bra med mönster eller helfärgad. Bara eller täckta axlar går bra, men kan vara tips att ta med en sjal eller kofta då det kan vara kallt i kyrkan. Translate ObsOBSobs
